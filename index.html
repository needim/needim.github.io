PROJECTS


a jquery notification plugin

NOTY is a jQuery plugin that makes it easy to create alert - success - error - warning - information & confirmation messages as an alternative the standard alert dialog. Each notification is added to a queue.

The API provides lots of other options to customise the text, animation, speed, buttons and much more.
It also has various callbacks for the buttons such as opening and closing the notifications and queue control.

customizable terminal

TERMIC is an idea for personal pages and terminal lovers You can add commands easily.
Terminal has a default "help" command for listing available commands and "clear" for the clean screen. Command autocomplete with "tab" key. Command history support for "up", "down" arrow keys.

If you want, you can able to create a blog with Termic. It's limited with your imagination.
